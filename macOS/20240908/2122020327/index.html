2024 新 Mac 購買後必裝的應用程式


這篇文章將介紹新 Mac 購買後建議安裝的應用程式，幫助您快速上手，提升生產力與開發效率。

一般使用工具

生產力工具

  1. Alfred 5 - 快速啟動與搜索工具
  2. Amphetamine - 防止 Mac 進入睡眠
  3. Bartender 5 - 管理菜單欄圖標
  4. Obsidian - 知識管理筆記應用
  5. Session - 番茄工作法計時器
  6. Spark - 電子郵件管理工具
  7. The Unarchiver - 檔案解壓縮工具

瀏覽器

  1. Arc - 新穎的瀏覽器
  2. Brave Browser - 隱私保護瀏覽器
  3. DuckDuckGo - 隱私瀏覽器
  4. Google Chrome、Google Chrome Canary - 網頁開發與測試
  5. Firefox、Firefox Developer Edition - 網頁開發瀏覽器
  6. Microsoft Edge - 微軟瀏覽器
  7. Vivaldi - 高度可自定義瀏覽器

設計與多媒體

  1. Canva - 簡單的圖像設計工具
  2. Figma - 網頁設計與協作工具
  3. Spotify - 音樂串流服務
  4. Telegram - 即時通訊工具
  5. LINE - 通訊與社交工具
  6. Notion - 筆記與任務管理

開發工具

版本控制與資料庫

  1. DBeaver - 資料庫管理工具
  2. Fork - Git 圖形化管理工具
  3. Stats - 系統監控工具

開發者工具

  1. Docker - 容器管理工具
  2. Visual Studio Code - 輕量級開發編輯器
  3. iTerm - 替代終端工具
  4. Warp - 現代化終端
  5. draw.io - 圖表和流程圖工具
  6. Dash - 開發文件查閱工具
  7. DevUtils - 多功能開發工具包
  8. AlDente - MacBook 充電控制工具
